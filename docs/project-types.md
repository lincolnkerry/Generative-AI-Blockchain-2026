# Project Types

Teams should choose one primary project type and may combine a secondary type if useful.

## 1. Personal AI Butler
Build a local-first assistant that helps with daily work, scheduling, search, summarization, writing, or automation.

## 2. Research/Study Copilot
Build an assistant that supports reading papers, generating notes, comparing sources, or managing course/research material.

## 3. Blockchain Agent or DApp Assistant
Build an agent that interacts with wallets, smart contracts, EVM development tools, token data, or blockchain workflows.

## 4. Privacy-Preserving AI Service
Build a system where sensitive data stays local or is protected by a clear privacy/security architecture.

## 5. Cost-Efficient AI Stack
Build a two-tier or hybrid stack that compares local models, APIs, caching, routing, or small/large model delegation.

## 6. Smartened OpenClaw/Agent System
Extend an agent workflow with memory, retrieval, tool use, reflection, planning, monitoring, or another Week 11 smartening method.

## Required choice statement

Each team README must include:

- Primary project type
- Target user/problem
- Why this project is useful in practice
- How it differs from existing big-tech assistants or platforms
